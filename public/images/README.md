# Memory photos

These `memory-1.svg` … `memory-6.svg` files are gradient **placeholders**.

To use real photos, drop in files with the same names (any image format works —
just update the extension in `src/components/Gallery.tsx` if you switch away
from `.svg`, e.g. to `.jpg`):

```
public/images/memory-1.jpg
public/images/memory-2.jpg
...
```

Square images (1:1) look best in the grid.
